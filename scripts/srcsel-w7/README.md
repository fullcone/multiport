# srcsel W7 bilateral validation helpers

Best-effort, one-time helpers used to produce the evidence in
[`docs/tailscale-direct-multisource-udp-phasew7-bilateral-real-network-validation.md`](../../docs/tailscale-direct-multisource-udp-phasew7-bilateral-real-network-validation.md).
They orchestrate a Windows ↔ remote Linux W7 run with a self-hosted
Headscale 0.28.0 control plane reached over an OpenSSH local
port-forward.

> These scripts are **not maintained tooling**. They reflect the
> commands that were actually run for the W7 evidence; expect to
> tweak them for any other environment.

## Prerequisites

- Linux host (Ubuntu 24.04 / 22.04 confirmed) reachable over SSH from
  the Windows host. The remote needs `apt`, `curl`, and `systemd`.
- Windows host with:
  - Python 3.11+ and `paramiko` (`pip install paramiko`).
  - OpenSSH client (`ssh.exe`, built into Windows 10/11/Server 2019+).
  - Native Go 1.26+ if you want to build the Windows binaries here.
- Build outputs in `<repo>/.w7-bins/`:
  - `tailscaled-srcsel`, `tailscale-srcsel` (Linux amd64)
  - `tailscaled-srcsel.exe`, `tailscale-srcsel.exe` (Windows amd64)

## Connection config (env vars)

All scripts read connection info from the environment:

| Env var | Purpose |
| --- | --- |
| `SRCSEL_W7_HOST` | remote host or IP (required) |
| `SRCSEL_W7_USER` | remote SSH user (default `root`) |
| `SRCSEL_W7_PASS` | remote SSH password (required if no key) |
| `SRCSEL_W7_KEY` | absolute path to a private key (preferred over password) |
| `SRCSEL_W7_AUTH_KEY` | headscale preauth key generated by step 4 |

Optional knobs (Windows-side `08-windows-up.ps1` only):

| Env var | Default |
| --- | --- |
| `SRCSEL_W7_BIN_DIR` | `<repo>/.w7-bins` |
| `SRCSEL_W7_LOG_DIR` | `<repo>/.w7-logs` |
| `SRCSEL_W7_STATE_DIR` | `<repo>/.w7-state` |
| `SRCSEL_W7_MODE` | `baseline` (also: `force`, `auto`) |
| `SRCSEL_W7_HOSTNAME` | `srcsel-windows` |
| `SRCSEL_W7_TS_PORT` | `41642` |
| `SRCSEL_W7_FIRST_RUN` | non-empty triggers `tailscale up` |

## Pipeline

```
01-recon.py                # read-only OS / network / toolchain probe
02-upload-binaries.py      # sftp Linux binaries to /usr/local/bin/
03-install-headscale.py    # download + apt-install headscale 0.28.0
04-start-headscale.py      # systemd start + create user + preauth key
                           # capture the auth key into SRCSEL_W7_AUTH_KEY
07-setup-key.py            # generate ed25519 + push pubkey (one time)
05-server-up.py            # bring up server tailscaled (mode-controlled)
                           # set _FIRST_RUN=1 in env on the very first run
                           # to invoke `tailscale up`
08-windows-up.ps1          # Windows-side: tunnel + tailscaled + up
06-server-metrics.py       # capture server-side metrics + log lines
```

The `SRCSEL_W7_MODE` env var on `05-server-up.py` and
`08-windows-up.ps1` selects between the three runs documented in
the W7 phase doc:

- `baseline` — no `TS_EXPERIMENTAL_SRCSEL_*` vars
- `force` — `TS_EXPERIMENTAL_SRCSEL_FORCE_DATA_SOURCE=aux`
- `auto` — `TS_EXPERIMENTAL_SRCSEL_AUTO_DATA_SOURCE=true`

(Both modes also set `TS_EXPERIMENTAL_SRCSEL_ENABLE=true` and
`TS_EXPERIMENTAL_SRCSEL_AUX_SOCKETS=1`.)

## Example end-to-end run

```powershell
# Windows PowerShell
$env:SRCSEL_W7_HOST = "1.2.3.4"
$env:SRCSEL_W7_PASS = "<initial password for paramiko>"

cd <repo>/scripts/srcsel-w7
python 01-recon.py
python 02-upload-binaries.py
python 03-install-headscale.py
python 04-start-headscale.py
# capture the printed hskey-auth-... value:
$env:SRCSEL_W7_AUTH_KEY = "hskey-auth-..."

python 07-setup-key.py
$env:SRCSEL_W7_KEY = "$env:USERPROFILE\.ssh\id_ed25519_srcsel-w7"

# --- baseline (no srcsel) -------------------------------------------------
$env:SRCSEL_W7_MODE = "baseline"; $env:_FIRST_RUN = "1"
python 05-server-up.py
Remove-Item Env:\_FIRST_RUN

$env:SRCSEL_W7_MODE = "baseline"; $env:SRCSEL_W7_FIRST_RUN = "1"
.\08-windows-up.ps1
Remove-Item Env:\SRCSEL_W7_FIRST_RUN

# --- forced-aux mode ------------------------------------------------------
$env:SRCSEL_W7_MODE = "force"
python 05-server-up.py
.\08-windows-up.ps1
python 06-server-metrics.py
& "<repo>\.w7-bins\tailscale-srcsel.exe" --socket=\\.\pipe\srcsel-w7 ping --tsmp --c=5 --timeout=10s 100.64.0.1

# --- automatic mode -------------------------------------------------------
$env:SRCSEL_W7_MODE = "auto"
python 05-server-up.py
.\08-windows-up.ps1
python 06-server-metrics.py
& "<repo>\.w7-bins\tailscale-srcsel.exe" --socket=\\.\pipe\srcsel-w7 ping --tsmp --c=5 --timeout=10s 100.64.0.1
& "<repo>\.w7-bins\tailscale-srcsel.exe" --socket=\\.\pipe\srcsel-w7 debug metrics |
    Select-String -Pattern "^magicsock_srcsel"
```
